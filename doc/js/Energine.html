    <static> resize(img, src, w, h, r)

    Resize an requested image. The attribute src of the img-tag will be build as follow: Energine.resizer + r + 'w' + w + '-h' + h + '/' + src
    Parameters:
    Name Type Argument Default Description
    img HTMLImageElement Image, that will be resized.
    src string Source of the original image.
    w number Width of the new image.
    h number Height of the new image.
    r string <optional>
    		 '' Special attribute. For example, if additional shrinking must be applied, in order to not cross requested width and height, r must be 'r'.
    Source:
    • share/scripts/Energine.js, line 289

    Energine.resizer = 'http://www.site.ua/resizer/';
Energine.resize($$('img')[0], 'images/img01.png', 100, 50);
