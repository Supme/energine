<abstract, protected> scroll(direction, scrollNTimes, isSelected)

The core method that scrolls the items. If one of the new visible items is not marked as active, this will set leftmost or rightmost item as active and after that an event will be fired, that the new item is set as active. If this method will be called with isSelected == true then the selection will be ignored.
Parameters:
Name Type Argument Default Description
direction number Defines the scroll direction. It can be 1 or -1 to scroll forward and backward respectively.
scrollNTimes number <optional>
 1 Defines how many scrolls must be done by one call of scrolling.
isSelected boolean <optional>
 false Indicates whether one of the new visible items is already selected.
