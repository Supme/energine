Energine\share\gears\QAL::modify (   $mode,
  $tableName = null,
  $data = null,
  $condition = null 
)

Execute simple modification (INSERT, UPDATE, DELETE) operation in the data base.

The operation mode defines by one of the following constants:

  1. QAL::INSERT - inserting;
  2. QAL::UPDATE - updating;
  3. QAL::DELETE - removing.

Data for operation mode QAL::INSERT and QAL::UPDATE is given by an array like

array(field_name => value)

Operation condition is given by:

  1. an array like array(field_name => value);
  2. a string of WHERE like 'field1 = 4 AND field2 = 8'.

Return values:

  • Mode QAL::INSERT:
    • last generated ID for field type AUTO_INCREMENT, or
    • true if such field is not exist in the table.
  • Mode QAL::UPDATE, QAL::DELETE:
    • true by success
  • false by execution error.

Energine\share\gears\QAL::select ( )

Execute simple SELECT-request and return the result.

Field names of $fields can be:

  1. an array of names;
  2. single name;
  3. true - all table rows wil be selected.

Selecting condition $condition is given by:

  1. an array like array(field_name => value);
  2. a sting of WHERE condition like 'field1 = 4 AND field2 = 8'.

The sort order is given by:

  1. an array like array(field_name => sort_order);
  2. a string of ORDER BY like 'field1 DESC, field2 ASC'.

Limit is given by:

  1. an array like array(offset, amount_of_rows);
  2. a string of LIMIT like '32'

true will be returned if the the result is empty.
