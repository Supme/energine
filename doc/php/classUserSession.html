static UserSession::start (   $force = false)

Start session.

Actually the session in Energine starts as continuation of already existed session (created in auth.php). If there are no information about the session in cookies or posts (there is an exception for flash uploader) then no session will start. For captcha we need to force to start session.

const UserSession::DEFAULT_PROBABILITY = 10

Default probability to call garbage collector.

It calculates as follows: DEFAULT_PROBABILITY / session.gc_divisor, where session.gc_divisor is 100 by defaults. Example: 10 / 100 means that garbage collector will called with probability of 10%.
