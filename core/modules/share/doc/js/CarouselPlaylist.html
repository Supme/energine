Class: CarouselPlaylist

CarouselPlaylist

new CarouselPlaylist(element, itemSelector)

Holds an playlist, that will be used by Carousel objects.
Parameters:
Name Type Argument Description
element string | Element Can be the id of an element in DOM Tree, or CSS Selector, or an Element that holds playlist's items. In case with CSS Selector it will get only the first element.
itemSelector string <optional>
 CSS Selector of the playlist's items. If this argument is not defined, then all children of the holder will be selected as playlist's items.
Author:
  • Valerii Zinchenko
Source:
  • Carousel.js, line 1497
Throws:
No items were found in the playlist.

string
Example

HTML container for playlist.

<div id="playlistID" class="playlist">
    <div class="item">item1</div>
    <div class="item">item2</div>
    ...
</div>
