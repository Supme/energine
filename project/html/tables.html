Примеры таблиц

Описание

Для описания таблиц применяется раздел файла content.css. Для таблицы по умолчанию обнулены отступы, границы и поля т. к. таблица может быть применена не по назначению - для визуального форматирования.

Существует форматирование для двух типов таблиц - простой и полосатой.

В открывающий тег таблицы можно включить любые атрибуты - они будут применяться только при отключенных CSS.
